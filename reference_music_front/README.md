# 1team-project-FE

## 최근 정리
- 커버 이미지: 모든 카드/상세/플레이어에서 `tracks/{id}/cover` 엔드포인트 사용.
- 홈/프로필 카드: 길이(있을 때만), 좋아요/다운로드, AI provider/model 표시. 카드 폭 제한으로 레이아웃 안정화.
- 홈 인기 섹션: 다운로드/좋아요 상위 트랙을 최대 4개씩 표시.
- 상세 페이지: 비슷한 트랙에도 메타 정보/좋아요/다운로드 표기, 다운로드 수 표기 추가.
- 플레이어바: 다운로드 시 새 탭 이동 대신 Blob으로 직접 저장(보안 개선).
- 업로드: 오디오 최대 5MB, 커버 최대 2MB; 커버 허용 형식 안내 추가(JPG/PNG/WEBP).
- .gitignore 정리: 루트/프로젝트 공통 불필요 파일, env, 빌드 산출물 무시.
- 문서/참고 파일 등 추적 해제 절차를 `git rebase&reset gitignore.md`에 메모.

## 기본 실행
```
npm install
npm run dev
```
